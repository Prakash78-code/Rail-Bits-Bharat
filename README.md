## 💻 How to Run Locally

Make sure Node.js and npm are installed.

```sh
# Clone the repository
git clone <YOUR_GITHUB_REPO_URL>

# Go to project folder
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm install

# Start development server
npm run dev
